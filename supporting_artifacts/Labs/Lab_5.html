Time Series Plot

Create a visualization of how weights vary for each genus over the duration of the study.

Show code 
ggplot(data = surveys,
       mapping = aes(x = year, y = weight, color = genus)
       ) +
  geom_point() +
  facet_wrap(~genus) + 
  theme(legend.position = "none") +
  labs(x = "", y = "", title = "Weight (gm) of Each Genus of Rodent throughout the Study")

Calculate and plot the mean weight for each year (for each genus).

Show code 
surveys |> 
  group_by(genus, year) |> 
  summarize(mean_weight = mean(weight)) |> 
  ggplot(mapping = aes(x = year, y = mean_weight, color = genus)) +
  geom_line() +
  labs(x = "", y = "", title = "Average Weight (gm) of Each Genus of Rodent throughout the Study")
`summarise()` has grouped output by 'genus'. You can override using the
`.groups` argument.

Captures over the week

Create a visualization of the number of rodents captured each day of the week.

Show code 
surveys |> 
  ggplot(mapping = aes(x = day_of_week, fill = day_of_week)) +
  geom_bar() +
  labs(x = "", y = "", 
       title = "Number of Rodents Captured on Each Day of the Week") +
  theme(legend.position = "none")

Change the order of the day of the week to go Monday through Sunday.

Show code 
surveys |> 
  ggplot(mapping = aes(
    x = fct_relevel(day_of_week, c("Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun")), 
    fill = day_of_week)) +
  geom_bar() +
  labs(x = "", y = "", 
       title = "Number of Rodents Captured on Each Day of the Week") +
  theme(legend.position = "none")

Plot the number of rodents captured between the week days and weekends.

Show code 
surveys |> 
  mutate(time_of_week = fct_collapse(day_of_week, 
                                     weekday = c("Mon", "Tue", "Wed", "Thu", "Fri"), 
                                     weekend = c("Sat", "Sun"))) |> 
  ggplot(mapping = aes(
    x = time_of_week, 
    fill = time_of_week)) +
  geom_bar() +
  labs(x = "", y = "", 
       title = "Number of Rodents Captured on Weekdays vs. Weekends") +
  theme(legend.position = "none")
